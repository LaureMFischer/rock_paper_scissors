# Rock Paper Scissors Game

To play the game locally, choose which strategy you want to play against:
`favorite`, `last` or `randomize`

For example:

`ruby play_rbs.rb favorite`
`Type 'r', 'p', or 's'`
> r
`I chose s. You win!`
`You won 1 times`
`You lost 0 times`
`We tied 0 times`

At any time, type `Done!` to quit the game.
